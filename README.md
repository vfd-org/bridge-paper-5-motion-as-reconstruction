# Bridge Paper 5: Motion as Reconstruction

This repository contains **Bridge Paper 5 (BP5)** of the *Vibrational Field Dynamics* (VFD) programme.

BP5 provides a precise and conservative clarification of how **motion** should be interpreted in stabilised geometric field theories. Rather than treating motion as primitive transport of an object through a background, the paper shows that apparent motion arises from **reconstruction-consistent persistence of stabilised configurations across reference frames**.
